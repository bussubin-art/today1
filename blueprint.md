
# 今日穿搭推荐应用

## 简介

此应用程序根据用户当前的地理位置获取天气数据，并根据温度和天气状况推荐合适的服装。

## 设计和功能

*   **自动地理位置:** 应用程序将请求用户允许访问其当前位置，以提供准确的天气数据。
*   **天气数据:** 使用 OpenWeatherMap API 获取实时天气信息，包括温度、天气状况（例如，晴、雨、云）和图标。
*   **服装推荐:** 根据天气数据，动态生成服装推荐。
*   **视觉设计:**
    *   **排版:** 使用“Google Fonts”中的“Playfair Display”和“Roboto”字体，营造出优雅而易读的外观。
    *   **颜色:** 采用清新现代的调色板。
    *   **图标:** 使用天气图标来直观地表示当前天气。
    *   **布局:** 采用居中、响应式的卡片式布局，在所有设备上都能提供出色的视觉效果。
*   **技术:**
    *   HTML、CSS、JavaScript
    *   ES模块
    *   Fetch API
    *   Web组件

## 当前任务

1.  **HTML (`index.html`):**
    *   设置基本的HTML结构。
    *   添加Google Fonts链接。
    *   创建一个主容器来容纳应用程序。
    *   添加`<weather-card>`自定义元素。

2.  **CSS (`style.css`):**
    *   为`body`和主要元素设计样式。
    *   实现卡片式布局。
    *   添加响应式设计媒体查询。

3.  **JavaScript (`main.js`):**
    *   创建`<weather-card>`Web组件。
    *   使用地理位置API获取用户位置。
    *   使用`fetch`从OpenWeatherMap API获取天气数据。
    *   根据天气数据实现服装推荐逻辑。
    *   动态更新DOM以显示天气和推荐。
